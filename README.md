# API MAROMBA STORE

Esta API foi desenvolvida para gerenciar produtos e realizar vendas em um sistema de ponto de venda.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Atualização de produtos
- Remoção de produtos
- Registro de vendas
- Consulta de histórico de vendas

## Requisitos

- Docker
- Node.js: versão 18
- Banco de Dados: postgres
